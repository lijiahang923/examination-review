### **考点详细解析**

您特别提到了 **Te形变化、动词、形容词、形容动词的肯定/否定/过去时** 以及 **疑问词**。以下是针对这些考点的归纳：

#### **1. 动词 (Verb) 的变化**
在初级阶段，最核心的是 **Masu形（礼貌体）** 的时态变化和 **Te形（连接/进行时）** 的变化。

**A. 时态变化 (Masu形)**
| 时态 | 形式 | 例词 (去/ikimasu) | 例词 (吃/tabemasu) |
| :--- | :--- | :--- | :--- |
| **非过去肯定** | ～ます | 行きます (ikimasu) | 食べます (tabemasu) |
| **非过去否定** | ～ません | 行きません (ikimasen) | 食べません (tabemasen) |
| **过去肯定** | ～ました | 行きました (ikimashita)| 食べました (tabemashita)|
| **过去否定** | ～ませんでした | 行きませんでした | 食べませんでした |
*注意：题目六中大量考察了根据时间词（昨日、明日）来决定使用 `ます` 还是 `ました`。*

**B. 动词 Te形 (て形) 变化规则**
虽然这份试卷主要考 Masu 形，但 Te 形是日语核心，用于“正在做(te iru)”、“请做(te kudasai)”、“做完A做B”等句型。

*   **一类动词 (五段动词):** 看词尾（辞书形）
    *   `う` `つ` `る` 结尾 $\rightarrow$ 促音 **って** (例：買う $\rightarrow$ 買って)
    *   `む` `ぶ` `ぬ` 结尾 $\rightarrow$ 拨音 **んで** (例：読む $\rightarrow$ 読んで)
    *   `く` 结尾 $\rightarrow$ **いて** (例：書く $\rightarrow$ 書いて) *特例：行く $\rightarrow$ 行って*
    *   `ぐ` 结尾 $\rightarrow$ **いで** (例：泳ぐ $\rightarrow$ 泳いで)
    *   `す` 结尾 $\rightarrow$ **して** (例：話す $\rightarrow$ 話して)
*   **二类动词 (一段动词):** 去掉 `る` + **て** (例：食べる $\rightarrow$ 食べて)
*   **三类动词 (不规则):**
    *   来る (kuru) $\rightarrow$ **来て (kite)**
    *   する (suru) $\rightarrow$ **して (shite)**

#### **2. 形容词与形容动词的变化**
这是试卷第三部分的重点。

**A. 一类形容词 (i-adj)** (以 `い` 结尾，如：高い、寒い)
| 时态 | 变化规则 | 例子 (寒い samui) |
| :--- | :--- | :--- |
| **肯定** | 原形 + です | 寒いです |
| **否定** | 去 `い` + **くない** です | 寒**くない**です |
| **过去肯定** | 去 `い` + **かった** です | 寒**かった**です |
| **过去否定** | 去 `い` + **くなかった** です| 寒**くなかった**です |
*特例：いい (好的) $\rightarrow$ よくない、よかった、よくなかった*

**B. 二类形容词 / 形容动词 (na-adj)** (如：静か、元気)
| 时态 | 变化规则 | 例子 (静か shizuka) |
| :--- | :--- | :--- |
| **肯定** | 词干 + です | 静かです |
| **否定** | 词干 + **ではありません** | 静か**ではありません** |
| **过去肯定** | 词干 + **でした** | 静か**でした** |
| **过去否定** | 词干 + **ではありませんでした**| 静か**ではありませんでした** |

#### **3. 疑问词 (Question Words)**
试卷中大量出现了疑问词填空，关键在于看**回答**。

*   **人:** **だれ** (dare) - 谁 / **どなた** (donata) - 哪位(礼貌)
*   **物:** **何** (nani/nan) - 什么
*   **地点:** **どこ** (doko) - 哪里
*   **二选一:** **どちら** (dochira) - 哪一个 (例：苹果和香蕉哪一个...)
*   **多选一:** **どれ** (dore) - 哪一个 (三个以上)
*   **原因:** **どうして** (doushite) - 为什么
*   **状态/意见:** **どう** (dou) - 怎么样 (例：味道怎么样？)
*   **样态修饰:** **どんな** (donna) + 名词 - 什么样的... (例：どんなスポーツ - 什么样的运动)

#### **4. 助词重点 (Particles)**
*   **は (wa):** 主题标记。
*   **が (ga):**
    *   疑问词做主语 (だれが...)
    *   表示能力、好恶的对象 (歌が好きです、韓国語が分かります)。
    *   逆接 (好吃**但是**贵)。
*   **で (de):**
    *   交通工具 (バスで - 坐巴士)。
    *   手段/语言 (日本語で - 用日语)。
    *   动作发生的场所 (駅で - 在车站)。
*   **に (ni):**
    *   移动的目的 (迎えに行きます - 去迎接)。
    *   存在的场所 (机の上に - 在桌上)。
    *   时间点 (10時に - 10点时)。
    *   给予/接受的对象 (友達に - 给朋友/从朋友)。
*   **を (o):** 动作的对象 (ご飯を食べる - 吃饭)。
*   **より (yori) / ほう (hou):** 比较句 (AはBより - A比B... / Aのほうが - A更...)。
